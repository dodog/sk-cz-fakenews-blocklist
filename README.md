# Slovak and Czech fake news blocklist

_Blocklist of Slovak and Czech websites with misleading, fake, fraudulent or propagandistic content._

The list of blocked websites is based on https://konspiratori.sk/zoznam-stranok and is compatible with [Pi-hole](https://github.com/pi-hole/pi-hole).
The data is updated every day from the list at https://konspiratori.sk and converted into a compatible format. 

**Add this link to your blocklist:**
```
https://raw.githubusercontent.com/dodog/sk-cz-fakenews-blocklist/refs/heads/main/hosts.txt
```
> Note: This list is not officially affiliated with the creators of konspiratori.sk.
